1. 檔案與資料夾存取

本應用程式的核心功能是讓使用者手動選擇裝置上的檔案或資料夾， 並建立本地 HTTP 分享服務。為此，本程式會要求 「所有檔案存取權限」(MANAGE_EXTERNAL_STORAGE)。

我們不會在背景未經授權掃描、上傳或修改您的檔案。 僅有您主動加入分享的檔案與目錄才會由分享伺服器存取。

2. 本地 LAN HTTP 伺服器與明文傳輸

分享服務僅在您主動點擊「啟動」後運作，並在您停止服務時終止。

注意： 本地分享使用 HTTP 明文傳輸，不提供 LAN 傳輸層加密。 同一網路上的其他裝置或人員理論上可能觀察到相關流量。 建議僅在可信任的私人區域網路中使用本機分享。

3. 臨時外網分享（Cloudflare Tunnel）

當您主動啟用「臨時外網分享」時，應用程式會連線至 api.trycloudflare.com 取得隨機公開連結， 並建立加密通道。

啟用後，資料不再僅於本地網路流通，相關流量可能經由 Cloudflare 基礎設施中轉。公開網址為臨時性，且不保證持續可用。

Easy Cloud HFS 為獨立開發產品，並非 Cloudflare 官方產品。 開發者不會收集或監控經由此功能傳輸的內容。 未啟用此功能時，不會建立該臨時外網通道。

4. 二維碼生成（QR Code）

二維碼完全在裝置本地計算生成。 本程式不會將您的分享網址、IP 位址或 QR 內容傳送至外部 QR API。

5. 廣告與第三方分析 SDK

本程式不包含第三方廣告 SDK 或行為分析 SDK。 我們不收集遙測資料，也不會將 App 使用行為自動傳送給開發者。

6. 設定、密碼與 Session Token 儲存

密碼以明文儲存於 App 私有沙盒設定檔 easycloudhfs.json 中。

Session Token（登入 Cookie）會傳送至瀏覽器客戶端； 在 LAN HTTP 模式下，這些資訊會隨 HTTP 明文流量傳輸。 伺服器停止後，Session 與臨時 Token 會失效或清除。

本程式不會故意將您的密碼輸出至應用程式日誌。

7. 日誌與活動紀錄

本地下載紀錄可能包含時間、客戶端 IP、檔名與傳輸量等資訊， 並儲存於 App 私有儲存空間，僅供 App 內顯示與管理。

這些紀錄不會自動傳送給開發者，也不會自動上傳至開發者伺服器。 您可在 App 內清除相關紀錄。

8. 資料保留與刪除

停止伺服器時，相關 Session 與臨時 Token 會被清除或失效。 解除安裝 App 後，Android 通常會移除 App 私有資料。

您原本位於共享儲存空間中的檔案，不會因解除安裝 App 而自動刪除。 本 App 不建立線上使用者帳號，因此不提供「刪除帳號」功能。

9. 第三方元件

本應用程式使用第三方開源元件，並依其各自授權條款使用。 相關授權資訊可於 App 或專案文件中查看。

1. File and Folder Access

The core purpose of the app is to let users manually select local files or directories and make them available through a local HTTP sharing server. To provide this core functionality, the app requests All Files Access (MANAGE_EXTERNAL_STORAGE).

We do not perform unauthorized background scanning, uploading, or modification of your files. Only files and directories you explicitly choose to share are accessed by the sharing server.

2. Local LAN HTTP Server and Cleartext Traffic

The sharing server operates only after you manually start it and stops when you stop the service.

Warning: Local LAN sharing uses HTTP cleartext traffic and does not provide transport-layer encryption for LAN traffic. Other parties on the same network could theoretically observe this traffic. Use local sharing only on trusted private networks.

3. Temporary Public Link (Cloudflare Tunnel)

When you manually enable Temporary Public Link, the app contacts api.trycloudflare.com to obtain a random public URL and establish an encrypted tunnel.

Once enabled, traffic is no longer limited to the local network and may be relayed through Cloudflare infrastructure. The public URL is temporary and continued availability is not guaranteed.

Easy Cloud HFS is independently developed and is not an official Cloudflare product. The developer does not collect or monitor the content transmitted through this feature. No temporary public tunnel is created unless you enable it.

4. QR Code Generation

QR codes are generated entirely on your device. The app does not send your sharing URLs, IP addresses, or QR content to an external QR API.

5. Advertising and Third-Party Analytics SDKs

The app does not include third-party advertising SDKs or behavioral analytics SDKs. We do not collect telemetry and do not automatically send app usage behavior to the developer.

6. Settings, Passwords, and Session Tokens

Passwords are stored in plaintext in the app's private sandboxed configuration file, easycloudhfs.json.

Session tokens (login cookies) are sent to browser clients. In LAN HTTP mode, this information is transmitted as part of cleartext HTTP traffic. Sessions and temporary tokens are invalidated or cleared when the server stops.

The app does not intentionally print your password to application logs.

7. Logs and Activity Records

Local download records may include timestamp, client IP address, filename, and transfer size. These records are stored in app-private storage for in-app display and management.

They are not automatically sent to the developer or uploaded to a developer-operated server. You can clear the relevant records from within the app.

8. Data Retention and Deletion

When the server stops, related sessions and temporary tokens are cleared or invalidated. Uninstalling the app normally removes app-private data under Android's standard behavior.

Original files in shared storage are not automatically deleted when the app is uninstalled. The app does not create online user accounts and therefore does not provide a Delete Account feature.

9. Third-Party Components

The app uses third-party open-source components under their respective license terms. Related license information is available in the app or project documentation.
